# 👋 Intro 
안녕하세요. 5년차 JAVA, PHP 개발자 안민성입니다.<br>
에드테크·이커머스·Saas 등 다양한 도메인에서<br>
프론트, 서비스 설계, 유지보수, API 개발, 데이터 처리 중심으로 개발 경험을 쌓았습니다.<br>
특히 에드테크 - 광고 운영, 플랫폼 개발 분야에 강점이 있습니다.<br>

- 같은 일을 반복하지 않도록 구조적으로 효율화된 업무 방식을 선호합니다.
- 충분한 기획과 정의 이후 실행에 들어가는 프로세스를 선호합니다.
- Jira, Slack 기반으로 투명하게 공유되고 추적 가능한 협업 환경을 선호합니다.
- 개발 전 탄탄한 설계가 선행되는 방식을 선호합니다.
- 단순하고 유지보수 가능한 구조(비레거시, 비스파게티)를 선호합니다.
- 기본기에 기반해 복잡도를 통제하는 설계를 선호합니다.
- 역할과 책임이 명확하게 정의된 팀 구조를 선호합니다.
- 상급자의 도메인 이해와 기초 지식을 바탕으로 의사결정하는 리더십을 선호합니다.
- 상급자가 팀원의 역할과 책임, 숙련도를 명확히 파악하고 관리하는 리더십을 선호합니다.

위 기준이 충족되지 않는 환경에서는 생산성을 내기 어렵다고 판단하며, 그러한 방식의 업무는 지양합니다.
